Installation
============

To install serviceping, simply:

.. code-block::

    $ pip install serviceping

or using easy_install:

.. code-block::

    $ easy_install serviceping

or from source:

.. code-block::

    $ python setup.py install


Getting Started
===============

.. code-block:: python

    >>> import serviceping
